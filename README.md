# 壹默接口项目说明
## 1.模块组成
1.1 yimoapi-backend-parent #项目后端
1.2 yimoapi-fronted #项目前端
1.3 yimoapi-sdk #api-sdk

前端:
Vue、Vite、View Design、Pinia、Axios
